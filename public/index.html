3. Advanced Micro Adjust Logic

The Hidden Software Trap: Micro Adjust behaves differently based on the chosen mode type:
  • In ECO and TURBO (Static Modes): Adjusting shifts the single active Assist Level up or down instantly.
  • In AUTO and TRAIL (Dynamic Range Modes): The controller changes ONLY the upper bound value. If your Auto mode is set to Lvl 4-6, raising it moves the ceiling to 7, 8, or 9 while the floor stays at 4.
  • Critical warning: If you push the upper value below the floor limit during an active ride, you will permanently degrade the minimum baseline. This cannot be fixed via Micro Adjust on the fly and requires an app reset.
